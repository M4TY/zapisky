# Kvadratické rovnice s parametrem

parametr - p 

px2 + 2(p - 1)x + p - 5 = 0

## Pokud dosadíme 0 - rovnice není kvadratická

    0x2 + 2(0 - 1)x + 0 - 5 = 0
    -2x = 5
    x = -5/2

## Pomoci diskriminantu

    ax2 + bx + c = 0

    D = b2 - 4ac
